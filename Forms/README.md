## Para simular um back-end:

### Instale o JSON Server:
```
npm install -g json-server
```

### Navegue para:
```
cd Forms\src\app\demos\todo-list
```

### Execute:
```
json-server todo.db.json
```

### Acesse:
```
http://localhost:3000/
http://localhost:3000/todolist
```
